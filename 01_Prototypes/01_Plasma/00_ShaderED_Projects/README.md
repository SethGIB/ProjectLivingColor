
# Living Color

***

## Plasma Previsualization 
Standalone GLSL shader tests for the plasma efffects. These effects drive the LED colors that the client sends to the server

***

## Dependencies

The projects are presented as  `.sprj` files, which require [SHADERed](https://www.shadered.org), but the GLSL code (`.vert` and `.frag` files) can be used in any environment that loads GLSL. Note that you'll probably have to change some of the uniform names to match the names used in your specific environment.

***

## Customizing and Extending

These are pretty simple, so there's not too much to do here specifically.

- Try messing around with how each texture's UVs are generated by passing different expressions to `uv_0`, `uv_1`, `uv_2`, and `uv_3`
- Try using different blending modes for the noise samples. I got all mine from [jamieowen/glsl-blend](https://github.com/jamieowen/glsl-blend). It can also be helpful to do some previz in photoshop or affinity to get an idea for how different noises blend together based on different blending modes
